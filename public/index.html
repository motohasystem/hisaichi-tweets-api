被災地のつぶやき

被災された方々の声を届けるAPI

API Endpoints

  • GET /api/articles 記事一覧（page, limit, search, year, month, category）
  • GET /api/articles/random ランダム1件（?category=カテゴリ名）
  • GET /api/articles/:id 記事詳細
  • GET /api/categories カテゴリ一覧と件数
  • GET /api/stats 統計情報
